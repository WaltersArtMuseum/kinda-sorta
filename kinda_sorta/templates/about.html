About
Kinda-Sorta is an alternative approach to resource discovery within a museum's collection. Rather, than ask the user what they are looking for, it starts with a single object and asks the user what aspects that object are important or significant to them. They can adjust how important those facets are and it will then suggest similar objects using those criteria.

The inspiration for this project came from previous work on Serendip-o-matic as well as discussions with colleagues about 'user driven relevency'.
